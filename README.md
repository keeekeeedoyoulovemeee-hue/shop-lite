# ShopLite — интернет-магазин

## Структура проекта
- `frontend/` — фронтенд (HTML, React, Angular)
- `backend-java/` — микросервис для товаров (Spring Boot)
- `backend-python/` — микросервис для корзины (FastAPI)
- `gateway/` — API Gateway (Node.js)
- `database/` — SQL-скрипты для PostgreSQL

## Как запустить фронтенд
1. Открой `frontend/shop.html` в браузере
2. Магазин работает с mock-данными
